# Architecture Diagrams

This folder contains system design diagrams in Excalidraw format (`.excalidraw` files).

## Files

- **c4-context.excalidraw** — High-level context (actors, external systems)
- **component-diagram.excalidraw** — Internal components and dependencies
- **data-flow.excalidraw** — How data moves during run execution
- **run-lifecycle.excalidraw** — State machine for run execution
- **deployment-architecture.excalidraw** — Docker Compose topology

## How to Edit

1. **Online:** Upload `.excalidraw` file to [excalidraw.com](https://excalidraw.com)
2. **Desktop App:** Open with [Excalidraw app](https://excalidraw.com/app) (also available locally)
3. **VS Code:** Install "Excalidraw" extension

## How to Export

Once edited:
- **PNG:** File → Export → PNG (save to `rendered/` folder)
- **SVG:** File → Export → SVG
- **JSON:** File → Save (already in `.excalidraw` format)

## Rendered Images

PNG/SVG exports live in `rendered/` for quick viewing without opening the editor.
